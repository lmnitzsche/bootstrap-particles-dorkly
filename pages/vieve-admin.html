Create new articles for the geekiest publication around

How to Use

  1. Fill out all the fields below with your article information
  2. Click "Generate Article Code" to create the formatted JavaScript object
  3. Copy the generated code using the "Copy to Clipboard" button
  4. Open vieve-data.js and paste the code into the vieveArticles array
  5. Save the file and your new article will appear on the site!
